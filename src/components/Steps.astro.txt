        <p>Las credenciales son claves y tokens que permiten que OpenCode se conecte a Slack, Google Calendar y Google Sheets.</p>
      </div>

      <h3>Paso 6.1: Crear carpeta de configuracion</h3>
      <pre>mkdir -p "$env:USERPROFILE\.config\opencode"

      <ol>

      <ul>

      </ul>
      <ul>
        <li><strong>Variable name:</strong> <code>SLACK_TEAM_ID</code></li>
        <li><strong>Variable value:</strong> tu Team ID (empieza con <code>T</code>)</li>
      </ul>
      <ul>
        <li><strong>Variable name:</strong> <code>SLACK_APP_TOKEN</code></li>
        <li><strong>Variable value:</strong> tu App-Level Token (empieza con <code>xapp-</code>)</li>
      </ul>

      <ul>

      </ul>
      <ul>
        <li><strong>Variable name:</strong> <code>GOOGLE_SERVICE_ACCOUNT_KEY_FILE</code></li>
        <li><strong>Variable value:</strong> <code>C:\Users\TU_USUARIO\.config\opencode\google-sheets-service-account.json</code></li>

  <section class="step" data-step="7">
    <div class="step-header">
      <div class="step-icon">7</div>
      <div class="step-info">
        <span class="step-label">Paso 7</span>
        <h2>Crear Slack App</h2>
      </div>
    </div>
    <div class="step-content">
      <div class="info-box">
        <h3>Que es una Slack App?</h3>
        <p>Una Slack App es una aplicacion que permite que otras herramientas se comuniquen con tu workspace de Slack.</p>
      </div>

      <h3>Paso 7.1: Crear la app</h3>
      <ol>
        <li>Ve a: <a href="https://api.slack.com/apps" target="_blank">https://api.slack.com/apps</a></li>
        <li>Click en <strong>"Create New App"</strong></li>
        <li>Selecciona <strong>"From scratch"</strong></li>
        <li>En <strong>"App Name"</strong>, escribe: <code>MCP Server</code></li>
        <li>Selecciona tu workspace de Slack</li>
        <li>Click <strong>"Create App"</strong></li>
      </ol>

      <h3>Paso 7.2: Agregar permisos (OAuth Scopes)</h3>
      <p>En <strong>"OAuth & Permissions"</strong>, agrega estos permisos:</p>
      <ul>
        <li><code>channels:history</code></li>
        <li><code>channels:read</code></li>
        <li><code>chat:write</code></li>
        <li><code>reactions:write</code></li>
        <li><code>users:read</code></li>
        <li><code>users.profile:read</code></li>
      </ul>

      <h3>Paso 7.3: Instalar la app en tu workspace</h3>
      <p>Click en <strong>"Install to Workspace"</strong> y luego <strong>"Allow"</strong>.</p>

      <div class="warning-box">
        <h4>IMPORTANTE:</h4>
        <p>Guarda el <strong>Bot User OAuth Token</strong> (empieza con <code>xoxb-</code>) y el <strong>Team ID</strong> (empieza con <code>T</code>).</p>
      </div>

      <h3>Paso 7.4: Habilitar Socket Mode</h3>
      <ol>
        <li>Ve a <strong>"Socket Mode"</strong> y activa Enable Socket Mode</li>
        <li>Genera un App-Level Token con el nombre <code>MCP Server Token</code></li>
        <li>Agrega el scope <code>connections:write</code></li>
        <li>Copia el token (empieza con <code>xapp-</code>)</li>
      </ol>

      <h3>Paso 7.5: Suscribir a eventos</h3>
      <p>En <strong>"Event Subscriptions"</strong>, habilita eventos y agrega:</p>
      <ul>
        <li><code>app_mention</code></li>
        <li><code>message.channels</code></li>
        <li><code>message.groups</code></li>
        <li><code>message.im</code></li>
        <li><code>message.mpim</code></li>
        <li><code>reaction_added</code></li>
        <li><code>reaction_removed</code></li>
      </ul>

      <h3>Paso 7.6: Habilitar interactividad</h3>
      <p>En <strong>"Interactivity & Shortcuts"</strong>, activa Interactivity y usa cualquier URL HTTPS como Request URL.</p>
    </div>
  </section>

      <h3>verify-slack.ps1</h3>
      <pre># verify-slack.ps1
$token = [Environment]::GetEnvironmentVariable("SLACK_BOT_TOKEN", "User")
$teamId = [Environment]::GetEnvironmentVariable("SLACK_TEAM_ID", "User")

if (-not $token -or -not $teamId) {
    Write-Host "[ERROR] SLACK_BOT_TOKEN o SLACK_TEAM_ID no configurados" -ForegroundColor Red
    return
